カスタム専用メニュー v0.1

目的：
- 衣装と背景を「同じ入口」で選べるようにする
- サムネUIを見やすくする（枠 / 選択ハイライト / 名前）

操作：
- 上部ボタン
  🎨 : カスタムメニューの開閉
  ✨ : 背景タブで開く
  👗 : 衣装タブで開く

中身：
- タブ式（衣装 / 背景）
- グリッド表示
  - 背景：サムネ＋名前（ファイル名）
  - 衣装：暫定（👗アイコン＋名前）

実装メモ：
- legacy の WardrobeMenu / BackgroundMenu は触らず、別実装（game/custom_menu.py）
- サムネは make_theme_thumbs の出力を g._custom_bg_thumbs に保持して利用
